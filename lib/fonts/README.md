# Fonts for Watermark

Pro watermark používáme TTF fonty.

## Fallback font:
Pro základní funkcionalitu použijeme GD built-in fonty (imagettftext vyžaduje TTF).

## Instalace custom fontů:
1. Stáhni TTF font (např. Liberation Sans, Arial, atd.)
2. Ulož do této složky: `lib/fonts/Arial.ttf`
3. Font bude automaticky dostupný v watermark nastavení

## Výchozí fonty:
- LiberationSans-Regular.ttf (fallback, pokud není k dispozici)

**Poznámka:** Pro produkci doporuču přidat skutečné TTF fonty.
